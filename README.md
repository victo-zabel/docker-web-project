# Проект: Автоматизация сборки веб-приложения с Docker и CI/CD

## Описание

Этот репозиторий является частью моего портфолио и демонстрирует базовые навыки Junior DevOps инженера по автоматизации процессов сборки (Continuous Integration - CI).

**Цель проекта:** Взять простое статическое веб-приложение, упаковать его в Docker-контейнер с использованием Nginx и настроить GitHub Actions для автоматической сборки образа при каждом коммите в ветку `master`.

Это мой **второй учебный проект**, направленный на понимание логики работы Docker и принципов CI/CD.

## Используемые технологии

*   **Docker:** Контейнеризация приложения (`Dockerfile`).
*   **Nginx:** Веб-сервер, используемый внутри контейнера.
*   **GitHub Actions:** Оркестрация и автоматизация процесса сборки (CI/CD Pipeline).

Для запуска приложения на вашем компьютере (при установленном Docker):

# Перейдите в папку с Dockerfile
cd auto-server-project

# Соберите образ
docker build -t my-nginx-app .

# Запустите контейнер и откройте в браузере http://localhost:8080
docker run -d -p 8080:80 my-nginx-app
